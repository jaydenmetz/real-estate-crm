Test API Key Setup

This page helps you set up a test API key to bypass authentication issues temporarily.

Instructions:

  1. Click "Set Test API Key" to store the test key in localStorage
  2. The CRM application will automatically use this key for API requests
  3. Navigate back to the CRM and try accessing escrows
  4. Click "Remove API Key" when you want to go back to normal authentication

Note: This is a temporary debugging tool. The test API key provides limited access for testing purposes only.
