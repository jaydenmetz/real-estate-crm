How Open Graph Protocol Works

What happens when you share this page:

  1. When someone pastes this URL in iMessage, Slack, Facebook, etc.
  2. That service's server makes a single HTTP request to fetch this HTML
  3. It reads only the meta tags in the <head> section
  4. It creates a preview card using:
    • Title: "123 Main St, Los Angeles, CA - Beautiful 4BR Home"
    • Description: "Stunning 4 bedroom, 2.5 bath home..."
    • Image: The property photo URL

Key Points:

  • This is NOT scraping - you're providing this data intentionally
  • Services only read what you explicitly provide in meta tags
  • It's a one-time fetch when the link is first shared
  • The image is usually cached by the service

For Your Real Estate CRM:

You can add these tags to your property detail pages so when agents share listings, they get beautiful preview cards with property photos automatically.
