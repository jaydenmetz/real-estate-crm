## 🎨 COLOR CODING LEGEND

### Status Chip Colors
- **[ACTIVE]** - Blue gradient (`#3b82f6` → `#2563eb`)
- **[PENDING]** - Orange gradient (`#f59e0b` → `#d97706`)
- **[CLOSED]** - Green gradient (`#10b981` → `#059669`)
- **[SCHEDULED]** - Purple gradient (`#8b5cf6` → `#7c3aed`)
- **[HOT LEAD]** - Red gradient (`#ef4444` → `#dc2626`)
- **[QUALIFIED]** - Purple gradient (`#8b5cf6` → `#7c3aed`)

### Badge Indicators
- **[TODAY]** 🔴 - Red badge with pulse animation
- **[SOON]** 🟡 - Yellow badge for < 24 hours
- **⭐** - Star icon for hot leads (score ≥ 75)
- **👁️** - Eye icon for visibility toggle
- **[Delete ×]** - Appears on hover (top-right)

### Progress Bar Colors
All progress bars use the status color gradient:
- Escrows: Green (checklist completion)
- Listings: Blue (days on market relative to average)
- Clients: Purple (pipeline stage progress)
- Appointments: Orange (time until appointment)
- Leads: Red/Orange/Grey (lead score temperature)

## 📱 RESPONSIVE BEHAVIOR

### Desktop (>1200px)
```
Small:  [Card 320px]
Medium: [Card 320px] [Panel 380px]
Large:  [Card 320px] [Panel 380px] [Panel 240px] [Panel 240px]
```

### Tablet (600-900px)
```
All modes show 2 panels at a time with navigation arrows:
[← Panel 1] [Panel 2 →]
```

### Mobile (<600px)
```
All modes show 1 panel at a time with carousel:
[← Panel →]
● ○ ○ ○  (panel indicators)
```
